\# CineLog



CineLog is a comprehensive media tracking application designed to help users manage their watchlist, track episode progress, and organize their personal library of movies and series.



\## Features

\- \*\*Centralized Tracking:\*\* Keep all your movies and series in one place.

\- \*\*Progress Management:\*\* Track exactly which episodes you've watched.

\- \*\*Cross-Source Data:\*\* Integration with APIs (Jikan/TMDB) for rich media information.

\- \*\*Personal Library:\*\* Easily categorize your media as Watching, Completed, or Plan to Watch.



\## Tech Stack

\- \*\*Backend:\*\* Node.js, Express

\- \*\*Database:\*\* PostgreSQL (Neon)

\- \*\*Frontend:\*\* React (Coming soon)

\- \*\*Deployment:\*\* Git



\## Folder Structure

\- `/client` - Frontend application.

\- `/server` - Backend API and business logic.

\- `/db` - Database schema definitions.



\## Getting Started

1\. Clone the repository: `git clone <your-repo-url>`

2\. Install dependencies in `/server`: `npm install`

3\. Set up your `.env` file with `DATABASE\_URL`.
